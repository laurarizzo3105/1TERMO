# 🎓 Resumo Integrado: Tecnologia e Desenvolvimento de Sistemas

Este guia consolida os conceitos fundamentais abordados nas disciplinas de Arquitetura IoT, Sistemas Operacionais, Programação e Engenharia de Software.

---

## 🏗️ 1. Arquitetura IoT & Hardware (Arduino)
A base da Internet das Coisas conecta o mundo físico ao digital.
- **Hardware:** Uso de microcontroladores (Arduino) para leitura de sensores e controle de atuadores.
- **Linguagem C++:** Desenvolvimento de firmware (baixo nível) focado em eficiência e controle de GPIOs.
- **Conectividade:** Integração com Python para processamento de dados e comunicação via Serial/MQTT.

---

## 🖥️ 2. Sistemas Operacionais (Linux/Debian)
O ambiente onde as aplicações ganham vida e os recursos são gerenciados.
- **Sistemas Abertos vs. Fechados:** Diferenças entre o modelo colaborativo (Linux) e o proprietário (Windows/macOS).
- **Distribuições:** O papel do **Debian** como base estável para servidores e IoT.
- **Terminal (CLI):** Comandos essenciais de navegação, permissões e gestão de pacotes (`apt`).

---

## 🐍 3. Lógica de Programação & Boas Práticas
O desenvolvimento de software moderno exige organização e qualidade.
- **Python:** Linguagem versátil para lógica de programação, automação e integração de sistemas.
- **Clean Code:** Escrita de código legível, nomes significativos e funções modulares.
- **Git/GitHub:** Controle de versão e colaboração técnica essencial para o mercado.

---

## 📋 4. Engenharia de Requisitos
A fase crucial para garantir que o sistema atenda às necessidades reais.
- **Tipos de Requisitos:**
  - **Funcionais:** O que o sistema faz.
  - **Não Funcionais:** Desempenho, segurança e restrições.
- **Técnicas de Elicitação:** Brainstorming, entrevistas e prototipagem para validação rápida.
- **Documentação:** Criação de diagramas UML e relatórios técnicos para formalizar a arquitetura.

---

## 🔄 5. A Visão Sistêmica (O Elo)
Um projeto completo de tecnologia une todos esses pontos:
1. O **Levantamento de Requisitos** define o que será feito.
2. O **Sistema Operacional** provê a infraestrutura necessária.
3. A **Lógica em Python e C++** constrói as funcionalidades.
4. A **Arquitetura IoT** permite que o software interaja com o mundo real.
