# 🐍 Lógica de Programação com Python

Este guia estabelece os fundamentos de programação, ferramentas de colaboração e padrões de escrita de código para o curso.

---

## 📚 1. Conteúdo de Aula: Fundamentos de Python
*Aprender a sintaxe e a estrutura de pensamento computacional.*

- **Tipos de Dados e Variáveis:** `int`, `float`, `string`, `bool`.
- **Estruturas de Decisão:** `if`, `elif`, `else`.
- **Estruturas de Repetição:** `while` e `for` (iteração em listas e ranges).
- **Coleções:** Listas, dicionários e tuplas.
- **Funções:** Definição, parâmetros e retorno de valores.

---

## 🐙 2. Controle de Versão: Git & GitHub
*Como gerenciar o código e trabalhar em equipe.*

- **Conceitos de Git:**
  - `git init`: Iniciar um repositório.
  - `git add .`: Preparar arquivos (staging).
  - `git commit -m "mensagem"`: Salvar o estado atual.
- **Integração com GitHub:**
  - Criação de repositórios remotos.
  - `git push` e `git pull`: Sincronizando o trabalho.
  - O uso do `.gitignore` para projetos Python.

---

## ✨ 3. Clean Code (Código Limpo)
*Escrevendo códigos que humanos conseguem ler.*

- **Nomes Significativos:** Variáveis e funções devem dizer a que vieram (ex: `soma_total` em vez de `s`).
- **Funções Pequenas:** Uma função deve fazer apenas uma coisa e fazê-la bem.
- **Comentários Necessários:** O código deve ser autoexplicativo; use comentários apenas para o "porquê", não para o "como".
- **DRY (Don't Repeat Yourself):** Evitar repetição de código através de funções e classes.
- **PEP 8:** Seguir o guia de estilo oficial da linguagem Python.

---

**Projeto: Gerenciador de Tarefas via Terminal**

2. Aplicar os conceitos de **Clean Code** na organização das funções.
